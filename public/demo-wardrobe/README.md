# Demo Wardrobe Images

This directory should contain high-quality clothing images for the demo page.

## Image Requirements

- **Format**: JPG or PNG
- **Size**: 400x400px minimum (square aspect ratio)
- **Style**: Flat lay photos or ghost mannequin shots
- **Background**: Neutral/white backgrounds
- **Lighting**: Consistent, professional lighting
- **Naming**: Use descriptive names matching the wardrobe items

## Recommended Images

1. `white-tee.jpg` - White t-shirt
2. `black-blazer.jpg` - Black blazer

## Image Sources

You can source images from:
- Unsplash (https://unsplash.com) - Free, high-quality photos
- Pexels (https://pexels.com) - Free stock photos
- Looka (https://looka.com) - AI-generated images
- Your own product photography

## Current Implementation

The demo currently uses Unsplash URLs as placeholders. To use local images:

1. Add images to this directory
2. Update `app/demo/page.tsx` to use `/demo-wardrobe/filename.jpg` instead of Unsplash URLs

Example:
```tsx
image: "/demo-wardrobe/white-tee.jpg"
```
